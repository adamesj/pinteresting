# One Month Rails

This is a pinterest clone application from [*One Month Rails*](http://onemonthrails.com).

The application is part of Startup Institute's Web Development orientation.

Topics covered in this tutorial include:

- Setting up a programming environment
- Deploying your application to Github and Heroku
- Configuring routes
- Installing the 'bootstrap-sass' gem followed by including @import 'bootstrap' to the main CSS file.
- Using the devise gem to add users and authorization.
- Using the 'masonry-rails' gem to make the pins responsive.
